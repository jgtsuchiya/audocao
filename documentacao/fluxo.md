1. Tela Inicial
    - Ao abrir o aplicativo, o usuário é recebido com a tela inicial que apresenta as principais funcionalidades e opções de navegação.

2. Login/Registro
    - O usuário pode optar por fazer login com suas credenciais existentes ou registrar uma nova conta.

3. Registro
    - O usuário poderá de registrar como doador (instituição ou pessoa física) ou como adotante.

4. Registro doador
    - O usuário deve fornecer informações básicas, como nome, e-mail, telefone e endereço.
    - O usuário deve fornecer informações adicionais dependendo do tipo de doador escolhido:
        - Para instituições: nome da instituição, CNPJ, área de atuação.
        - Para pessoas físicas: CPF, data de nascimento.
    - O usuário deve escolher o tipo de doador (instituição ou pessoa física).
    - O usuário deve criar uma senha segura para sua conta.

5. Após feito o registro o usuário é redirecionado para um tela que informa que o registro foi realizado com sucesso e que será analisado para aprovação.
